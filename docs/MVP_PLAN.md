MVP Plan (4 sprints)

Sprint 1 (1-2 weeks):
- Custom User model + auth (admin, teacher, student roles)
- Student CRUD with photo upload
- Simple admin dashboard and teacher/student dashboards
- Basic API endpoints for students and users

Sprint 2 (2-3 weeks):
- Classes & Subjects management
- Attendance recording (web + API)
- Exam scheduling and marks entry

Sprint 3 (2-3 weeks):
- Fees management (invoices, payments status)
- Library (books checkout/returns)
- Notifications (email/console)

Sprint 4 (2 weeks):
- Role-based dashboards with summaries and charts
- Production hardening (Postgres, Docker, static/media handling)
